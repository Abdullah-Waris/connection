Connection — Social Journaling App

Connection is a React Native (Expo) mobile app where users create daily posts, track streaks, manage reminders, and browse an archive of past entries. The app supports light/dark mode, uses global context for shared state, and includes a working backend built with Node.js, Express, Prisma, and SQLite.

⸻

Features
•	Create daily posts with text, time, and optional images
•	Track posting streaks
•	View past posts by month/year in the Archive
•	Manage group reminders with time pickers
•	Switch between light and dark themes in Settings
•	Clean UI with theme-aware components
